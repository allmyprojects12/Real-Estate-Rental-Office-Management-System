# Real-Estate-Rental-Office-Management-System
Real-Estate-Rental-Office-Management-System

​وصف المشروع:​اسم المشروع: نظام إدارة مكتب تأجير عقارات
​النوع: نظام لإدارة الأعمال المالية والإدارية متكامل (Management Information System - MIS)، يهدف إلى تطوير النظام القائم لمعالجة القصور الهيكلي في الأتمتة المالية والأمن وتكامل البيانات الإدارية.
​الوظيفة الأساسية: يركز النظام على أتمتة العمليات المالية المعقدة، وأهمها تطوير آلية لإنشاء جدول أقساط شهرية آلياً عند توقيع كل عقد إيجار. كما يهدف إلى تعزيز الأمان والمساءلة عبر إنشاء سجل تدقيق (Audit Log) لتتبع أنشطة المستخدمين، وتوفير وحدة حسابات وتقارير مالية جديدة لحساب صافي الأرباح والخسائر وأداء الوسطاء بشكل دوري. بالإضافة إلى ذلك، يعمل على تحسين إدارة المحتوى عبر إضافة إمكانية ربط وتخزين صور العقارات.
